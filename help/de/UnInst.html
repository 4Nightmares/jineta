Deinstallieren

Zum Deinstallieren von jineta gehen Sie wie folgt vor:

  • Starten Sie jineta mittels java -jar jineta.jar -u
    Dies löscht die Einträge in der Registry o.ä.
  • Löschen Sie die Datei jineta.jar und, falls vorhanden, auch die Datei jineta.errors

Das war's dann auch schon.
